# Design-of-air-pollution-monitoring-system
通过物联网检测大气污染浓度我使用了 Arduino uno 作为设计的开发板，然后我可以构建 MQ-135 气体传感器的功能（识别空气中污染气体浓度的能力）。利用 LCD1602 可直接观察和使用物联网远程监控，实现对气体浓度变化和报警反应的远程监控。 这不仅快，而且安全。当人们看到这种有害气体的浓度如此之高，就会更加关注 大气质量。同时也可以做出一些改变，创造一个良好的工作环境，毕竟，这是人 们生存所需要的。
一开始用 MQ-135 型气体传感器测定当前位置有害气体浓度（PPM）的变化 值；
再然后我既然选择来 Arduino，那自然要用它来编写代码能够实现一些我想 要的功能；
接下来在 LCD1602 屏幕上可以直接观察到有害气体的浓度； 紧跟着 ESP8266 将接收到的值传输到物联网； 在最后如果有害气体浓度过高，整个时间段都会发出警报。
当这里的空气质量太低了，污染的 ppm 的值超过了规定的范围，那么用舵机 来控制的设备（比如门窗等）会通过舵机的转动而打开，那么就可以对气体进行 交换
